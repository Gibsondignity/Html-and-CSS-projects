Assigning Values to Variables

Python is a dynamically typed language. This means that you don't even need to declare the type of variable while assigning a value to a variable in python. Other programming languages like C++, C, JavaSccript, Java, among others, there is a rigorous decclaration of variables before assigning values to them. We use the equal to sign (=) to assign values to variables. 

num1 = 100              # An integer number
num2 = 100.00           # A floating point number 
name = "Gibson"         # A string 

print(num1)
print(num2)
print(name)

Here we assigned 100 to num1, 100.00 to num2, and Gibson to name. The print() function prints the specified message to the screen, or other standard output device. The message can be a string, or any other object, the object will be converted into a string before written to the screen.

Output: 
100
100.00
Gibson

Data Types

Data types are the classification or categorization of data items. The data stored in memory can be of many types. For example, a person's age is stored as a numeric value and his or her address is stored as alphanumeric characters. Python has various standard data types that are used to define the operations possible on them and the storage method for each of them.

Common Data Types
Integers -2, -1, 0, 1, 2, 3, 4, 5
Floating-point numbers -1.25, -1.0, --0.5, 0.0, 0.5, 1.0, 1.25
Strings 'a', 'aa', 'aaa', 'Hello!', '11 cats'

Standard Data Types

Python has five standard or built-in data types
  • Number
  • Sequence Type
  • Boolean
  • Set
  • Dictionary

Number

Number data type in python represent the data which has numeric value. Numeric value can be integer, floating number or complex numbers These values are represented as int, float or double and complex class in python.

Integers
This value is represented by int class. It contains positive or negative whole numbers (exclude fraction or decimal). In python there is no limit to how long an integer value can be.
Float
This value is represented by float class. It is a real number with floating point representation. It is specified by a decimal point. Optionally, the character e or E followed by a positive or negative integer may be appended to specifiy scientific notation.

Strings

Creating String

Strings in Python can be created using single quotes or double quotes or even triple quotes.

Let's do some MADLIP Game. I recommend you type this in the Integrated Development Environment (IDE) we downloaded during the installation process for better understanding 

name = input("Enter your name: ")
home_town = input("Enter your  home town: ")
my_stay = input("Enter where you stay now: ")
age = input("Enter your age: ")
color = input("Enter your body color: ")
nose_shape = input("Enter your nose shape: ")

print(f"My name is {name}")
print(f"I come from {home_town} but stay at {my_stay }")
print(f"I am  {age} years of age")
print(f"I am  {color} in color")
print(f"I have {nose_shape} nose!")

Here we are taking input from the user using the input() function. The input() function presents a prompt to the user and waits for the user's responds. The input from the user is read as a string and is assigned to a variable.The python interpreter then insert the input from the user to replace the variables in the parenthesis and then print out the actual input entered by the user. The "f" before the string and {} in the print function are called f-Strings, a new and improved way to format Strings in python
