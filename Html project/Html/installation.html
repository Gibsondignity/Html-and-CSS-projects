Python Installation on Windows, Mac OS X and Other platforms

Python is a widely used high-level programming language first launched in 1991. Since then, Python has been gaining popularity and is considered as one of the most popular and flexible server-side programming languages. Unlike most Linux distributions, Windows does not come with the Python programming language by default. However, you can install Python on your Windows server or local machine in just a few easy steps.

Step 1: Download Python Executable Installer

Head on to Python.org which is the official page for python. Click on Downlads. As you can see from the picture below, python supports different operatins systems such as Windows, Linux/UNIX, Mac OS X and Others. Check which opearating system you are using. In our case we are downloading for Windows opearating system;
Looking for Python with a different OS? Python for Windows, Linux/UNIX, Mac OS X, Other Click on Windows to be sure of the OS type. Click on the download button to start the download process.



NB: Note that python 3.9.0 cannot be used on Windows 7 or earlier. So if you are using Windows 7 for now you would have to download Version 3.8.0 or below.



Step 2: Run The Python Executable Installer

Head on to Python.org if not yet downloaded the python installer.
Run the Python Installer once downloaded. Make sure you tick the Install launcher for all users (recommended) and Add Python 3.9.0 to PATH checkboxes. The Add Python 3.9.0 to PATH places the interpreter in the execution path.
Click on Install Now to begin the installation process – the recommended installation options.



Step 3: Installation In Progress

Head on to Python.org if not yet downloaded the python installer.
Wait while the installation is still in progress. Make sure you don't click on the cancel or the close [X] button bellow and above respectively.



Step 4: Successful Installation Message

Head on to Python.org if not yet downloaded the python installer.
After the installation process, a sucessful message has to appear indicating that the installer has been installed sucessfully. Click on the close button as the setup has been sucessfully installed on your computer.

Head on to Python.org if not yet downloaded the python installer.
